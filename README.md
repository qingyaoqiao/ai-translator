# 🌍 AI 全能文档翻译机 (AI Document Translator)

这是一个基于 Node.js + Python + DeepSeek/OpenAI API 的全能文档翻译工具。
支持保留 Word/PDF 的格式（表格、图片、字体样式），并提供简洁的网页界面。

## ✨ 功能特点
- **多格式支持**：支持 .txt, .docx, .pdf。
- **格式保留**：采用 XML 式替换，最大程度保留 Word 原文格式。
- **极速并发**：支持多线程并发翻译，加大长文档处理速度。
- **智能 PDF**：利用 Python `pdf2docx` 将 PDF 转换为可编辑文档后翻译。

## 🛠️ 技术栈
- **后端**：Node.js (Express), Python (pdf2docx)
- **前端**：HTML/CSS (原生)
- **AI**：OpenAI 接口 (推荐硅基流动的 DeepSeek，便宜好用，也是本翻译器默认选择)

## 🚀 如何使用

本项目提供两种使用方式：网页端直接使用和本地文件部署。

### 网页端
您可以直接访问我们的在线网站（https://qyq.zeabur.app），无需任何本地配置，即可上传文档并进行翻译。

### 本地部署
如果您希望在本地环境中运行此翻译器，请遵循以下步骤：
1.  克隆或下载本仓库代码。
2.  确保您的计算机已安装并配置好 Python 环境。
3.  安装所需的 Python 依赖库，尤其是 `pdf2docx`。
4.  根据指引启动 Node.js 后端服务。
5.  在浏览器中打开相应地址即可使用。

## ⚠️ 注意事项

*   **隐私与安全**：我们非常重视您的数据安全。本网站**不会以任何形式保存**您的翻译原文、翻译结果以及您填写的 API Key。所有翻译进程均在临时的会话中完成，页面关闭后所有数据即被清除。
*   **运行环境**：
    *   **网页端**：无需任何特殊配置，有浏览器即可自由使用。
    *   **本地部署**：需要在您的计算机中正确安装和配置 **Python** 环境，否则将无法处理 `.pdf` 文件的翻译请求。
*   **API Key**：本工具需要您提供自己的 DeepSeek 或 OpenAI API Key 进行翻译。费用将由对应的 AI 服务提供商收取，本工具自身不收取任何费用。
*   **翻译质量**：翻译结果由您选择的 AI 模型生成，其质量取决于模型的语言能力。在处理专业性强或语境复杂的文档时，建议对结果进行人工校对。
